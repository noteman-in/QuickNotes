QuickNotes Privacy Policy

Last Updated: July 2026

QuickNotes stores notes locally on the user's device using Chrome's chrome.storage.local API.

Information We Collect

QuickNotes does not collect, transmit, sell or share any personal data.

Website Content

Highlighted text is saved only when the user explicitly chooses to save it. The data remains stored locally on the user's own browser.

Third Parties

QuickNotes does not transmit user data to external servers.

Contact

For bug reports or questions, please use the official feedback form.
